# **🥣 Программный комплекс «Исследуем Сметану» (ResearchSmetana)**

Добро пожаловать в репозиторий передового программного обеспечения, созданного для решения важнейшей проблемы человечества — глубокого, всестороннего и бескомпромиссного математического моделирования и исследования **Сметаны**.

Забудьте про нейросети и блокчейн. Настоящие инновации происходят здесь.

## **🧐 О проекте**

**ResearchSmetana** — это высокотехнологичное десктопное приложение, которое предоставляет уникальный инструментарий для Администраторов и Исследователей. Программа позволяет взаимодействовать с базой данных сметаны (Smetana.db), проводить строгие математические расчеты (MathModel.cs) и наслаждаться непревзойденным пользовательским опытом (UX), вдохновленным золотой эрой интернета и операционной системой Windows XP.

### **✨ Ключевые (и фатальные) особенности**

* **Многопользовательская архитектура:** Разделение прав доступа на «Исследователей» (ResearcherInterface) и «Администраторов» (AdminInterface). У каждого свой путь к познанию молочнокислого продукта.  
* **Математическая модель:** Встроенный модуль MathModel.cs для расчетов, суть которых понятна только автору (и, возможно, самой сметане).  
* **Иммерсивный звуковой дизайн:** Интегрированы классические звуки Windows XP (windows-xp-critical-error.wav, windows-xp-startup.wav). Ошибки программы будут причинять физическую боль, как и задумано.  
* **Авангардный UI/UX:** Кастомные курсоры (cursor.cur), анимированные предупреждающие колобки из ICQ (kolobok-warning.gif) и иконки пользователей высокой четкости.  
* **Встроенная симуляция отчаяния:** Наличие формы BSOD.cs гарантирует, что программа может (и будет) вызывать синий экран смерти просто потому, что может.  
* **Безопасность:** Кастомные диалоговые окна (Dialog.cs) и суровая валидация данных (CheckValues.cs), чтобы никто не смог испортить нашу базу данных сметаны.

## **🛠 Технологический стек**

* **Язык:** C\#  
* **Фреймворк:** Windows Forms (потому что WPF — это для зумеров)  
* **База данных:** SQLite (Users.db, Smetana.db)  
* **Ассеты:** Мемы образца 2010-х, звуки из 2001-го, колобки из 2005-го.

## **🚀 Установка и запуск**

Чтобы прикоснуться к великому, вам понадобится Visual Studio (любой версии, которая еще помнит, что такое WinForms).

1. Склонируйте этот проклятый репозиторий:  
   git clone https://github.com/hlebushek25/researchsmetana.git

2. Откройте IssleduemSmetanu.sln в Visual Studio.  
3. Убедитесь, что NuGet восстановил пакеты SQLite (см. packages.config).  
4. Нажмите F5 (или кнопку Start).  
5. Наслаждайтесь звуком запуска Windows XP. *Рекомендуется сделать звук колонок погромче.*

**ВНИМАНИЕ:** Проверьте пути к базам данных (Databases/Smetana.db и Databases/Users.db). Возможно, потребуется прописать относительные пути в InteractionDB.cs, если сметана откажется загружаться.

## **📜 Правила контрибьютинга (Стиль Коммитов)**

Если вы хотите внести свой вклад в исследование сметаны, пожалуйста, придерживайтесь нашего корпоративного стиля написания коммитов.

Примеры приемлемых коммитов:

* fix: сметана больше не вылетает при жирности \> 20%  
* feat: добавил еще больше звуков ошибок виндовс, чтобы пользователи страдали  
* refactor: математическая модель теперь учитывает фазу луны при расчете густоты  
* style: колобок теперь крутится быстрее при критической ошибке БД  
* docs: добавил этот шикарный ридми
